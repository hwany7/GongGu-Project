# GongGu-Project(ver_1 / 2021-0803)

1. 프로젝트 개요
- 공동구매 기능을 제공하는 웹사이트 제작
- 스프링 레거시 프로젝트 + 직접 .jar 파일 추가 방식 사용 (원시적)
- MyBatis, Oracle RDBS 사용

2. 개발 버전
- Java SE Development Kit 8 241
- Apache Tomcat 9.0
- Oracle Database 11g Express Edition
