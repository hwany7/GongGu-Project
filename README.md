# GongGu-Project

1. 프로젝트 개요
- 공동구매 기능을 제공하는 웹사이트 제작
- 스프링 레거시 프로젝트 + 직접 .jar 파일 추가 방식 사용 (원시적)
- MyBatis, Oracle DBMS 사용
