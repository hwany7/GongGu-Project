# GongGu-Project

1. 프로젝트 개요
- Spring Web MVC을 활용한 공동구매 기능을 제공하는 웹사이트 제작
- 메이븐을 활용하지 않고 직접 .jar 파일을 추가하면서 구현
- 본 프로젝트는 능력내에서 빠르게 프로젝트를 완성하는 것에 중점을 둠
